# Enclosure

## Definition

The Enclosure is the tanks, cabinets, ports, bulkheads, couplers, sealing architecture, and physical system boundary that define the closed biosphere.

## Current State

- miniBIOTA is a fully closed working concept, but airtight closure is not yet achieved.
- Minor passive air leakage remains.
- Six 29-gallon biome tanks form the main ecological sequence.
- Four vertical 29-gallon atmosphere tanks sit above the terrestrial/coastal biomes.
- Custom plywood cabinets create intentional elevation differences.
- Connections use drilled 3 in ports, Fernco rubber PVC couplers, rigid PVC sections, and bottom/top/side bulkhead routing.
