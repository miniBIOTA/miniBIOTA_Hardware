# Rain Manifolds

## Current Role

Rain manifolds distribute released condensate from the cloud reservoirs into the biomes. They are the Rain System hardware downstream of condensation and reservoir tipping.

## Current Documentation State

The active repo currently has only high-level rain manifold context. The exact current manifold design, dimensions, outlet counts, and per-biome routing still need to be documented.
