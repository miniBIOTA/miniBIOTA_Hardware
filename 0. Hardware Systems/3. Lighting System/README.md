# Lighting System

## Definition

The Lighting System is the photoperiod and spectrum control that drives photosynthesis across all biomes.

## Current Documentation State

Detailed Lighting System hardware is not yet documented in this repo.

Future biome-specific lighting hardware should be recorded in the relevant biome folder. Shared lighting standards, fixture families, controller architecture, photoperiod strategy, and safety limits should be recorded here.

## System Boundary

Included here:

- Lighting fixtures.
- Spectrum and intensity standards.
- Photoperiod control.
- Shared timers/controllers.
- Future ESP32 lighting integration.

Not included here:

- Plant/ecology interpretation of lighting response, which belongs with Research/Ecosystem context.
- Biome-specific installed fixture details, which belong in that biome's hardware file once known.

## Open Items
