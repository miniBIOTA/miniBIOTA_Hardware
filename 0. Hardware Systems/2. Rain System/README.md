# Rain System

## Definition

The Rain System is the cloud reservoirs, rain manifolds, condensate routing, and gravity-driven release hardware that collect water condensed by the Climate System and distribute rainfall into the biomes.

## Current Deployment

- The four existing atmosphere tanks each contain cloud reservoirs.
- Each atmosphere has four triangular prism reservoirs, called clouds.
- Condensate forms on chilled rear glass, flows downward, and collects in cloud reservoirs.
- Reservoir tipping is gravity-driven.
- Observed cadence is roughly every 2-3 weeks, depending on climate settings and ambient conditions.

## System Boundary

Included here:

- Cloud reservoirs.
- Reservoir pivots and bearings.
- Rain manifolds and condensate distribution paths.
- Gravity release behavior.

Not included here:

- Heat exchangers, chiller, coolant, or pump control; those belong to the Climate System.
- Tanks, ports, and sealing; those belong to Enclosure.
