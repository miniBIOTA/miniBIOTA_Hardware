# Hardware Memory Inbox

Temporary holding area for candidate durable memory before it is promoted to a named Hardware memory file.

## Triage Rules

- Use this file only when a durable fact or rule has no obvious home yet.
- Keep entries short and source them to the session or file that introduced them.
- Promote stable items into `memory/01-*` through `memory/08-*` when their home is clear.
- Delete or archive temporary items once resolved.
